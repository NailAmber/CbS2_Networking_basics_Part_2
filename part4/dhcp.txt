Что означает вывод “DDORA IP ….” при DHCP запросе - каким DHCP-пакетам из пакетной трассы DORA соответствует?
	
Вывод "DDORA ..." при DHCP запросе - это индикация процесса DHCP: Discover -> Offer -> Request -> Ack.
Соответствия пакетам: D -> Discover, O -> Offer, R -> Request, A -> Ack.


На какой IP отправляется первый DHCP-пакет?

Первый DHCP-пакет отправляется нв широковещательный адрес - 255.255.255.255.
